# 🍕 Pizza Sales SQL Analysis

## 📌 Project Overview
This project focuses on analyzing pizza sales data using SQL to uncover meaningful business insights.  
The goal is to understand sales performance, customer preferences, and revenue trends.

---

## 🛠 Tools Used
- SQL (MySQL)

---

## 📊 Analysis Performed
- Total Revenue Calculation  
- Top Selling Pizzas  
- Category-wise Sales Analysis  
- Order Trends (Daily/Monthly)  
- Quantity Sold Analysis  

---

## 📈 Key Insights
- Identified top-performing pizza categories  
- Found peak sales periods and order trends  
- Analyzed contribution of each category to total revenue  
- Helped understand customer ordering behavior
